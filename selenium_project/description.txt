---Описание тестов:

(test_login.py)
test_success_login
Тестирует успешный вход в систему с корректными учетными данными.
Ожидаемый результат: Отображение приветственного сообщения.

test_unsuccess_login
Тестирует вход с некорректными учетными данными (неправильный логин или пароль).
Ожидаемый результат: Появление сообщения об ошибке, информирующего о неверных данных для входа.


(test_registration.py)
test_success_registration
Проверяет успешную регистрацию новой учетной записи.
Ожидаемый результат: Появление сообщения об успешной регистрации и доступ к аккаунту.
test_unsuccess_registration
Проверяет, что повторная регистрация с уже существующим логином вызывает сообщение об ошибке.
Ожидаемый результат: Отображение сообщения о том, что логин уже используется.

(test_search.py)
test_search_with_results:
Выполняет поиск по существующему ключевому слову, ожидая, что отобразятся результаты (например, хотя бы один элемент с классом result_item).
test_search_no_results:
Выполняет поиск с ключевым словом, для которого нет результатов, проверяя, что появляется сообщение об отсутствии результатов (например, элемент с ID no_results_message).
